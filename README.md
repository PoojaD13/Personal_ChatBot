# - JARVIS AI Agent 
# --An intelligent document analysis system with multi-format file support, powered by AI and semantic search.

# -Features
-Multi-Format Document Support: PDF, Word, Excel, PowerPoint, Text, Images<br>
-OCR Capabilities: Extract text from images and scanned documents<br>
-Semantic Search: AI-powered document understanding and retrieval<br>
-Visual Content Analysis: Process and analyze images with vision models<br>
-Real-time Q&A: Ask questions about your uploaded documents<br>
-Vector Database: Efficient document storage and retrieval using Pinecone<br>

# -Quick Start
-Prerequisites<br>
-Python 3.8+<br>
Ollama (for local LLM models)<br>
Pinecone account (for vector database)<br>
