# - JARVIS AI Agent 
# --An intelligent document analysis system with multi-format file support, powered by AI and semantic search.

# -Features
-Multi-Format Document Support: PDF, Word, Excel, PowerPoint, Text, Images
-OCR Capabilities: Extract text from images and scanned documents
-Semantic Search: AI-powered document understanding and retrieval
-Visual Content Analysis: Process and analyze images with vision models
-Real-time Q&A: Ask questions about your uploaded documents
-Vector Database: Efficient document storage and retrieval using Pinecone

# -Quick Start
-Prerequisites
-Python 3.8+
Ollama (for local LLM models)
Pinecone account (for vector database)
